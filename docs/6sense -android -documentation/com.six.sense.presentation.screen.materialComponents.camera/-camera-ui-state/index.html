CameraUiState

data class CameraUiState(val surfaceRequest: SurfaceRequest? = null, val cameraSelector: CameraSelector = CameraSelector.DEFAULT_BACK_CAMERA, val imageUris: List<Uri> = listOf(), val qrData: String? = null)

Represents the UI state of the camera screen.

This data class holds the necessary information to manage the camera's visual and functional state, including the active camera surface, camera selection, captured image URIs, and any detected QR code data.
