enum SystemInstructions : Enum<SystemInstructions>

Enum class representing different system instructions tailored for specific roles. Each enum value holds an instruction string and a role description.

These instructions are designed to guide the system in adopting a specific tone, language, and focus when interacting with users based on their profession or role. They provide a way to customize the system's response style to better meet the expectations and needs of different user personas.
