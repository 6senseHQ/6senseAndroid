DummyPagingSource

class DummyPagingSource : PagingSource<Int, DummyPaging>

A dummy implementation of PagingSource used for testing or demonstration purposes.

This class provides a minimal implementation of PagingSource that always returns a single page with a single DummyPaging item. It is not intended for use in production code where actual data pagination is required.

The load method always returns a page containing a single DummyPaging object. Both prevKey and nextKey are set to null, indicating that there are no previous or next pages.

The getRefreshKey method provides a basic implementation to calculate a refresh key based on the anchor position.
